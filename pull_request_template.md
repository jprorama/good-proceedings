Thank you for submitting to the GOOD Conference Proceedings!

Please name your PR with `<type>: <title>`, where `<type>` is either `Paper`, `Slides`, or `Poster` and `<title>` is the title of your submission. An editor will then add the `draft` label to your PR and GitHub Actions will run automated checks and create a preview build.

See the https://good-proceedings.curve.space/author-instructions for full submission instructions.
